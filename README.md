# Markdown Editor

A modern, Electron-based Markdown editor with a focus on aesthetics and ease of use.

## Features

- **Live Preview**: Real-time rendering of your markdown using the powerful `marked` library.
- **Format Toolbar**:
  - **Headings**: H1 through H6.
  - **Lists**: Bullet points, Numbered lists, and Task lists (`- [ ]`).
  - **Formatting**: Bold, Italic, Strikethrough, Quotes, and Code blocks.
- **Rich Media**:
  - **Images**: Insert images via URL or **browse your local files**.
  - **Links**: Easily insert links; external links open in your default browser.
  - **Tables**: Create tables with custom rows and columns.

## Installation
#### Snap:
`sudo snap install open-markdown-editor`

### Manual:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the application:
    ```bash
    npm start
    ```

## Development

Run with DevTools open:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```
